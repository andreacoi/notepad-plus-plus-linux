What's inside

Everything Notepad++ — Built for the Mac

All the features you rely on, reimplemented natively. No Wine, no Electron, no compromises.

⚡

Native Apple Silicon

Built for arm64. Runs at full speed with no Rosetta translation layer — instant launch, minimal memory.

🎨

48+ Languages

Syntax highlighting powered by Lexilla for C/C++, Python, JavaScript, Swift, Rust, Go, SQL, and dozens more.

🪟

Split View Editing

Open files side-by-side vertically or horizontally. Move or clone tabs between views. Sync scrolling.

🔍

Powerful Search

Find & Replace with regex, Find in Files, Incremental Search, Mark text with 5 styles, bookmarks.

📋

Side Panels

Function List, Document Map, Document List, and Clipboard History panels — just like Windows Notepad++.

🎭

Themes & Styles

Built-in dark/light themes and a full Style Configurator. Import NPP XML theme files directly.

⌨️

Macro Recording

Record, save, and replay macros. Run a macro multiple times. Full Scintilla-level keystroke fidelity.

💾

Session Restore

Auto-backup, session save/restore, tab pinning, and tab bar wrap mode. Your workspace, preserved.

🔧

Complete Menu Bar

All 377 menu items wired — case conversion, line sort, Base64, hashing, encoding, column editor, and more.
