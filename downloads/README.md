# Downloads

DMG releases are placed here. The website download button links directly to files in this folder.

To publish a new release, add the DMG here and push:

```bash
cp NotepadPlusPlusMac.dmg downloads/NotepadPlusPlusMac.dmg
git add downloads/NotepadPlusPlusMac.dmg
git commit -m "Release vX.X.X"
git push
```

The download URL used by the website is:
```
https://github.com/nppmss/notepad-plus-plus-macos/raw/main/downloads/NotepadPlusPlusMac.dmg
```
